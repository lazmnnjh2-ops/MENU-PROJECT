# ITNE352 Project (Group 3)

## Files
- `server.py` - Multi-threaded server handling clients
- `client.py` - Terminal client
- `gui.py` - GUI with CustomTkinter
- `cache.py` - Manages categories, areas, ingredients
- `api_client.py` - Fetch API data and save JSON
- `protocol.py` - Send/receive JSON messages
- `run_project.bat` - Run server + GUI
- `reference_Group3.json` - Cached reference data
- `saved_recipes.json` - Saved recipes
- `data/` - JSON storage folder
- `README.md` - This file

## Setup
```bash
pip install requests customtkinter
```

## Run
- With batch: `run_project.bat`
- Manually:
```bash
python server.py
python gui.py
```
